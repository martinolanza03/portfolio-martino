# Portfolio Martino

In questo repository trovi una raccolta dei miei progetti più significativi, sviluppati per migliorare le mie competenze e sperimentare nuove tecnologie. Ho incluso sia lavori personali che esercitazioni pratiche, con l’obiettivo di mostrare il mio approccio allo sviluppo, alla progettazione e alla risoluzione dei problemi.

Ogni progetto è accompagnato da una breve descrizione, dalle tecnologie utilizzate e, quando possibile, da una demo o da screenshot. Il portfolio è in continuo aggiornamento man mano che imparo e costruisco nuove soluzioni.

Se hai curiosità o suggerimenti, sei il benvenuto!
